# BMI Calculator - Flutter
A simple Flutter application that calculates the Body Mass Index (BMI) based on the user's height and weight. The app supports both Metric (cm/kg) and Imperial (in/lb) measurement systems, displays the BMI result with its health category, and allows users to save previous results in a history list.  Features Calculate BMI using Metric or Imperial units. Select gender. Enter height, weight, and age. Display BMI value and health status. Save BMI results to a history page. View previously saved BMI records. Input validation for empty fields. Built With Flutter Dart  This project was developed as a Flutter practice project to demonstrate UI design, navigation between screens, state management using StatefulWidget, and basic BMI calculation logic.
